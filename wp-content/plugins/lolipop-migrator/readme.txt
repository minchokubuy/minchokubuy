ロリポップ！レンタルサーバーの「WordPress簡単引っ越し」機能が利用するプラグインです。

== Description ==

ロリポップ！レンタルサーバーの「WordPress簡単引っ越し」機能が利用するプラグインです。

このプラグインを利用すると、別サーバーで運用中のWordPressをロリポップ！レンタルサーバーのサーバー上に引っ越しすることができます。


= 利用対象 =
ロリポップ！レンタルサーバーをご契約のお客様は、無料でご利用いただくことができます。

ご利用の際は、ロリポップ！レンタルサーバーのユーザー専用ページ「WordPress簡単引っ越し」機能から必要な情報を入力していただく必要があります。


= サポート =
プラグインの使用方法は、ロリポップ！レンタルサーバーのマニュアルをご参照ください。

ロリポップ！レンタルサーバー サポート 利用マニュアル
https://lolipop.jp/manual/


== Installation ==
注) このプラグインを使用するためには、ロリポップ！レンタルサーバーのいずれかのプランの契約が必要となります。

このプラグインは、直接インストールしていただく必要はありません。ロリポップ！レンタルサーバーの「WordPress簡単引っ越し」機能を利用いただいた際に、自動的にインストールされ、引っ越しが開始されます。
